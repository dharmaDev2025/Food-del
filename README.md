# Food Delivery System 

A full-stack **Food Delivery System** built with **MongoDB, Express.js, React.js, Node.js**, and integrated with **Stripe** for secure payment processing. This platform allows users to browse food items, place orders, and pay online. Admins can manage food items and view all orders.

---

## Features

### User Features
- Browse food items with images, descriptions, and prices.
- Add items to the cart.
- Place orders securely.
- Make online payments via **Stripe**.
- View order history.

### Admin Features
- Add, update, or delete food items.
- View all orders placed by users.
- Manage users and order statuses.

### Payment
- Integration with **Stripe Payment Gateway**.
- Secure and fast online transactions.

---

## Tech Stack

- **Frontend:** React.js, HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (with Mongoose)
- **Payment:** Stripe API
- **Authentication:** JWT (JSON Web Tokens)
- **State Management:** React Context API / Redux (if used)
- **Other Tools:** Axios, Nodemon, dotenv, bcrypt
